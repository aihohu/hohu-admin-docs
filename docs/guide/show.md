# 在线演示

我们提供了在线演示环境，让你无需安装即可体验 HoHu Admin 的完整功能。

## 访问地址

**演示地址**：[https://show.hohu.org](https://show.hohu.org)

## 登录账号

| 项目   | 值       |
| ------ | -------- |
| 用户名 | `admin`  |
| 密码   | `123456` |

::: warning 注意
演示环境的数据会定期重置，请勿输入真实敏感信息。
:::

## 功能预览

登录后你可以体验以下功能：

### 用户管理

- 用户的增删改查
- 用户状态启用/禁用
- 分配角色

### 角色管理

- 角色的增删改查
- 配置角色的菜单权限
- 配置角色的数据权限

### 菜单管理

- 树形结构的菜单配置
- 支持目录、菜单、按钮三种类型
- 动态路由生成

### 部门管理

- 树形结构的部门配置
- 按部门划分数据权限

### 系统监控

- 在线用户查看
- 操作日志审计
- 登录日志查询
